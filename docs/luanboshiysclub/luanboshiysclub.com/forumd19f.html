提醒：如果你不相信鬼魂存在、阴间存在，那么，请勿阅读这篇文章，以免看了气愤，影响体质。


传统中医说，肝藏魂。
人有肉体和灵魂。人在阳间。人去世后，灵魂变为鬼魂，生活在阴间。

从阴阳的角度说，阳间为阳，阴间为阴。两者，是互相影响的。-----“阴无所依，阳无所附”。当阴间乱了，最终必然会祸及阳间。

这么些年，到处讲科学论科学，然后，到处挖坟掘墓-----不管老坟新坟，到处平坟；所谓公墓，就是在地表弄个水泥的小坑；更多的骨灰就在多层的架子上放着；-----多少人家，不烧纸钱不祭奠；多少人家，一束鲜花就把祖宗打发了-----。

-----
几年前，目睹这种种对阴间的摧残，我就忧心忡忡，经常跟朋友们说，“阴无所依，阳无所附”，阴间乱了，必然会波及阳间。
几年来，鬼附身的疾病，越来越常见。以前最常见的是自己家的祖宗们找后代子孙，这几年，越来越多孤魂野鬼附身他人的病例。
-----

在轮回中，鬼魂投胎，是直接投在胎儿的体内。
胎儿，全程都在子的时期。
也就是：鬼魂进入阳间，在于：子。

同样的道理，阴间乱了，波及阳间，会在时间：子。

-----

这次的新冠肺炎，是在己亥年的丙子月发生；在庚子年开始愈演愈烈。
看公布出来的患者舌像，都说是湿，但事实上，那是：秽浊。
在很多地方，鬼魂被民间称为“脏东西”，在舌像上，就是体现为秽浊-----也就是看上去很脏。

这次新冠肺炎的性质，并非传统中医辨证的风寒暑湿燥火之类，而是来自阴间的那种鬼魂的阴寒之气。
阴间乱了许多年，在子这个时间，波及了阳间。

-----

子，        对应肾经。
足少阴肾经，“-----，其直行者，从肾上贯肝膈，入肺中，-----”。肾经有一段是直接入肺的。
在肾经上，负责管理在肺这段肾经的穴位，叫做：大钟穴。

多年前，我就讲过大钟穴。大钟，就是大的钟声。“丧钟为谁而鸣”，这句话大家都耳熟能详，却很少人想过其背后含义。其背后含义是：钟声，是开通黄泉路，让鬼魂进入阴间的。-----所以，我再三告诫亲朋好友，绝对不允许在家里挂那种发出声音的钟以及铃铛、风铃等类似钟的东西，以免不知不觉中开通了黄泉路，勾引鬼魂上来。

大钟，就是大的钟声，是人体内的黄泉路，是鬼魂进入人体的路。

有人睡觉总是反复做噩梦，梦到过世的祖宗们。我让贴两大钟穴，阻断体内的黄泉路，于是，不再做噩梦。

新冠肺炎的病毒性质是来自阴间的那种类似鬼魂的阴寒之气，它进入人体的通路，就是肾经上的大钟穴。
看影像学表现似乎是肺的问题，但实际上，病毒真正所在的人体空间方位，是肾经上大钟穴所管理的在肺的肾经区域。

-----

预防新冠肺炎：
1、        穴位取肾经上的两大钟穴。方法：取干的桑葚子，捣碎，贴大钟穴，医用胶布固定，两大钟穴都贴，早中晚各换一次。这种贴法，能够阻断体内的黄泉路，不让病毒进入人体。
2、        大碗吃米粥、米饭或者面食。
粮食，能壮大人的在阳间的气息，阻止鬼魂的阴寒之气靠近。
去年我看到一篇博客，大意是：某人去到神仙们（有鬼魂自然就有神仙）的一个官衙，有神仙关照，要好好吃饭。-----神仙强调要“好好吃饭”，显然是预测到了如今的情形。对付来自阴间的阴寒之气，阳间的粮食，尤其是主食，有着异乎寻常的药用效果。
3、        人参。
就是中国东北产的人参，各种名头的人参都算，包括生晒参或者白参、红参等，但不包括西洋参。
人参，顾名思义，成人形，于是，自然就具有驱除鬼魂的阴寒之气的效果。
4、中成药，推荐生脉饮。

大碗喝粥吃饭吃面食，大剂量给予人参水或者生脉饮，治疗可收奇效。

-----
关于新冠肺炎的消毒：

现在到处都是酒精、84消毒液，我是很怀疑它们的消毒效果的-----那么浓烈的气味，人在其中，吸入肺里，恐怕未等新冠病毒到来，各人的肺里先充满了酒精或者84消毒液的气息，肺先被刺激的病了。
还有艾灸。都知道抽烟伤肺，怎么艾灸的烟与火吸入肺内，就不担心伤肺了呢。艾灸的烟气火气，可是比香烟的烟气火气厉害的多了。

现在流行的消毒方式中，有一个提议是不错的。那就是用苍术。-----用苍术煮水，熏屋子或者浸泡衣物或者喷洒。
如上所述，新冠肺炎的病毒性质是阴间波及阳间的阴寒之气。而苍术，是传统上的驱除鬼气的药品，所以，用于这次新冠肺炎的消毒，很好。

-----

网上流传有许多药方，比如，广州那边八医院的一号方，据说治疗了几十例轻症患者，效果很好。
我注意过这些药方。那些效果较好的处方里，往往有：苍术、金银花、青蒿、白茅根等，这些药物，都是中医传统上的驱鬼气药物。所以，对于轻症患者，确实有效。-----这些药物其实可以做成香囊佩戴作为预防。
我还注意过朋友圈，有人患病后高烧不退，后来自己煮薏米水喝后热退。薏米，也是驱鬼气的食物，理论上确实有效。

-----

疫情发生以来，我每天都在盯着新闻。看到临床一线的医生穿着防护服，天天泡在汗水里，一方面对他们敬佩不已，感恩他们的付出；另一方面，也免不了忧心忡忡。
“汗为心之液”，每天这样大量出汗，心气大量损耗。而今年庚子年，无论庚还是子，对心气都大为压制。心气大量损耗的结果，轻则容易出现心脏问题，重则猝死。这些临床一线的医生，在以健康与生命来付出-----
中医的生脉饮，对于这种大量出汗影响心气，有较好的补充作用。这个药物其实很安全，可以大量喝。一盒一盒的喝通常都无妨。相信我的，去喝喝吧。

-----

现在人满脑子科学这科学那的，看到我这般分析，估计有人要气炸了，所以，初始我就提醒不信鬼魂阴间存在的别看。不多说了，信者得福！

愿我们的国家能够正视曾经和现在对阴间的祸乱，尽早让阴间阳间平衡和谐，让这次的疫情尽早过去。

祝福我的祖国！祝愿这次疫情尽早过去，尽早国泰民安！
